## Get started in three steps

<Steps>
  <Step title="Set up the backend">
    Start Docker services for PostgreSQL and MinIO, run database migrations, and launch the Go API server, WebSocket server, and worker process.
  </Step>
  <Step title="Configure the frontend">
    Install Node dependencies, set your Firebase and API credentials in `.env`, and start the Vite development server.
  </Step>
  <Step title="Create an account">
    Sign up as a buyer or seller at `http://localhost:5173`, then start browsing or posting listings immediately.
  </Step>
</Steps>

<Note>
  CampusHub requires Docker, Go 1.21+, and Node.js 18+ to run locally. See the [local setup guide](/deployment/local-setup) for full prerequisites.
</Note>
